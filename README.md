# Hospital Curita

Panel básico para gestionar citas médicas usando React + Vite y JSON Server como API local. Incluye login, dashboard, calendario, listado de citas y creación de nuevas citas. Maneja rutas protegidas y estado global con Context API.

## Funcionalidades
- Inicio de sesión

- Dashboard con estadísticas

- Crear / listar / eliminar citas

- Calendario visual

- UI estilo glass/iOS

- Rutas protegidas (AuthContext)

## Tecnologías

React, Vite, JSON Server, React Router, Context API, React Icons.

## Instalación

Clona el repo:
```
git clone https://github.com/tu-usuario/medical-panel.git
cd medical-panel

```

Dependencias  e inicio de backend

```
npm install
npx json-server --watch db.json --port 3001
npm run dev

```
